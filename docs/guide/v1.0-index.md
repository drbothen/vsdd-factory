# v1.0 Factory Plugin Kit — documentation index

> **Status:** index for v1.0-specific documentation. Filled in
> incrementally as Phase 0–5 stories ship; final shape locked at
> the 1.0.0 release gate (S-5.07).

The v1.0 line is the Rust dispatcher + WASM plugin rewrite. These
documents describe what changed, how to migrate, how to write hooks
against the new SDK, and how to run the observability pipeline. They
are separate from the existing v0.x docs in this directory because
v1.0 introduces new concepts (WASM plugins, registry-driven dispatch,
multi-instance sinks) that don't apply to v0.79.x.

## How to read this section while v1.0 is in flight

Each skeleton file in this index has a `Status:` banner indicating which
stories own which sections. Sections marked `TODO(S-X.Y)` are deliberately
empty until that story ships. The release gate at S-5.07 grep-checks
`docs/guide/` for any remaining `TODO(S-` markers and blocks the 1.0.0
tag if any are found, so doc completeness rides with code completeness.
