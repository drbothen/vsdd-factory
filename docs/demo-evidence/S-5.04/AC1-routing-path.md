## Dual-Routing-Tables Architecture

```
Claude Code runtime
      |
      | fires PostToolUseFailure
      v
hooks.json.<platform>          тЖР Layer 1 (BC-4.08.002)
  PostToolUseFailure entry
  command: .../factory-dispatcher
  async: true  timeout: 10000
      |
      | stdin: JSON envelope {event_name, session_id, dispatcher_trace_id, tool_input: {...}}
      v
factory-dispatcher binary
      |
      | reads hooks-registry.toml  тЖР Layer 2 (BC-4.08.003)
      | finds event="PostToolUseFailure" тЖТ plugin="hook-plugins/tool-failure-hooks.wasm"
      v
tool-failure-hooks.wasm
      |
      | calls emit_event("tool.error", {tool_name, error_message})
      v
Host fn: emit_event enriches with session_id, dispatcher_trace_id, plugin_name, plugin_version
         then appends construction-time fields (ts, ts_epoch, schema_version, type)
      v
Wire: 10-field tool.error event
```

File: `plugins/vsdd-factory/hooks/hooks.json.template`, line 111.

Key invariants confirmed:

- `timeout: 10000` (harness timeout; 10s > dispatcher budget 5s per ADR-011 hierarchy)
