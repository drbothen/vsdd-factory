## Anomalies

- **Homebrew rustc 1.94 shadowing:** The local environment has Homebrew cargo/rustc 1.94 earlier in PATH than rustup 1.95. All cargo commands were run by prepending `/Users/jmagady/.rustup/toolchains/1.95.0-aarch64-apple-darwin/bin` to PATH to ensure the correct compiler was used. This is a local-environment issue only; the worktree `rust-toolchain.toml` pins 1.95.0 correctly.
- **non_snake_case warnings in test files:** Test functions use BC/TV ID prefixes in uppercase (e.g., `test_BC_2_01_002_...`, `test_TV_001_...`). These produce `#[warn(non_snake_case)]` warnings during compilation. They do not affect correctness; all tests pass. The warnings are in test files (read-only for demo-recorder) and are a known style trade-off to preserve traceability IDs in test names.
