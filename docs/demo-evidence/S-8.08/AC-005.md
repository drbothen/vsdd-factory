## Integration Test Architecture

Tests invoke `factory-dispatcher` (not bare wasmtime) following the pattern from
`plugins/vsdd-factory/tests/regression-v1.0.bats`. The `VSDD_SINK_FILE` env var
routes plugin-emitted events to a tmpdir JSONL file for assertion. After dispatcher
execution, `jq` parses the JSONL to verify event fields.

```bash
run env CLAUDE_PLUGIN_ROOT="$PLUGIN_ROOT" CLAUDE_PROJECT_DIR="$WORK" VSDD_SINK_FILE="$SINK_FILE" \
  bash -c "printf '%s' '$envelope' | '$DISPATCHER'"
[ "$status" -eq 0 ]
run jq -r '.type' "$SINK_FILE"
echo "$output" | grep -q "agent.start"
```
