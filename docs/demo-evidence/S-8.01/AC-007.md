# AC-007 Evidence: host::emit_event Replaces bin/emit-event

**Story:** S-8.01 — Native port: handoff-validator (SubagentStop)
**AC:** AC-007 ([process-gap] traces to E-8 D-10 / R-8.07 architecture-compliance)

## Claim

`bin/emit-event` calls in the former bash script are replaced with
`host::emit_event` Rust calls in the native WASM crate. No reference to
`bin/emit-event` remains in `crates/hook-plugins/handoff-validator/`.
`bin/emit-event` binary is NOT removed (E-8 D-10 — deferred to S-8.29).

## Evidence

### No bin/emit-event references in crate

```
$ grep -r 'bin/emit-event' crates/hook-plugins/handoff-validator/
(no output)
```

Zero references to `bin/emit-event` in the handoff-validator crate.

### host::emit_event used exclusively

`src/main.rs` comment documents the substitution:
```rust
// Replaces `bin/emit-event` subprocess calls from the bash hook
// (AC-007 / E-8 D-10). `host::emit_event` is self-contained in
// the WASM guest; no subprocess or PATH lookup required.
vsdd_hook_sdk::host::emit_event(event_type, fields);
```

`src/lib.rs` module doc comment:
```
# AC-007 compliance note
`bin/emit-event` calls are replaced by `host::emit_event` here.
The `bin/emit-event` binary is NOT removed (E-8 D-10; deferred to S-8.29).
```

### host::emit_event call sites

Two call sites in `src/lib.rs` (via injectable `emit` callback in
`handoff_validator_logic`):

1. Empty result path (AC-003):
   ```rust
   emit("hook.block", &[
       ("hook",     "handoff-validator"),
       ("matcher",  "SubagentStop"),
       ("reason",   "subagent_empty_result"),
       ("severity", "warn"),
       ("subagent", agent),
   ]);
   ```

2. Short result path (AC-004):
   ```rust
   emit("hook.block", &[
       ("hook",       "handoff-validator"),
       ("matcher",    "SubagentStop"),
       ("reason",     "subagent_truncated_result"),
       ("severity",   "warn"),
       ("subagent",   agent),
       ("result_len", len_str.as_str()),
   ]);
   ```

`main.rs` wires the `emit` callback to `vsdd_hook_sdk::host::emit_event`.

### bin/emit-event binary NOT removed

```
$ ls -la bin/emit-event
-rwxr-xr-x ... bin/emit-event
```

`bin/emit-event` is present and unchanged. Removal is deferred to S-8.29
per E-8 D-10. AC-007 only replaces the subprocess calls in this hook's logic;
the binary itself is a shared utility and must not be removed prematurely.

### Forbidden dependency check

```
$ grep -i 'legacy-bash-adapter' crates/hook-plugins/handoff-validator/Cargo.toml
(no output)
```

No dependency on `legacy-bash-adapter` — forbidden per E-8 D-10.
The Cargo.toml only depends on `vsdd-hook-sdk` and `serde_json`.

## Pass / Fail

PASS — `host::emit_event` replaces all `bin/emit-event` subprocess calls from
the bash hook; zero `bin/emit-event` references in the crate; `bin/emit-event`
binary is untouched; no `legacy-bash-adapter` dependency.
