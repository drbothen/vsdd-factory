## Implementation (src/lib.rs lines 146-155)

```rust
pub fn is_blocked(result: &str) -> bool {
    use regex::RegexBuilder;
    let re = RegexBuilder::new(r"(?m)^(Status:|##?\s*)?\s*BLOCKED")
        .build()
        .expect("BLOCKED regex is valid");
    re.is_match(result)
}
```

Key design notes:
- Pattern: `(?m)^(Status:|##?\s*)?\s*BLOCKED` — multiline mode anchors `^` to line start
- Rust regex crate uses ERE-style alternation: `|` is unescaped alternation (not `\|`)
- Matches all variants: `BLOCKED`, `Status: BLOCKED`, `## BLOCKED`, `# BLOCKED`
- Does NOT match `BLOCKED` mid-line (e.g. `step completed but was BLOCKED by CI`)
