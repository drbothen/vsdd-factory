# AC-004: Pass/Fail Derivation Cascade

**Story:** S-8.09  
**BC Trace:** BC-7.03.073 postcondition 1 (status derivation cascade)  
**Status:** PASS

## What is demonstrated

The 5-step cascade:
1. `exit_code == 0` -> pass
2. `exit_code` non-zero, non-null -> fail
3. `interrupted == true` -> fail
4. `interrupted == false` -> pass
5. Neither determinable -> STATUS=unknown -> exit 0, no state write

Success path: exit_code=0 -> pass; exit_code=1 -> fail; interrupted=true -> fail; interrupted=false -> pass.  
Error path (EC-002): no exit_code + no interrupted -> unknown -> no state write, no event.
