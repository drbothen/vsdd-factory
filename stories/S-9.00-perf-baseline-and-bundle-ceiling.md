## Narrative

- **As a** W-16 wave gatekeeper preparing to dispatch 7 batch stories
- **I want to** know (a) the current WASM bundle sizes for all hook plugins post-rc.4,
  (b) a per-plugin bundle-size delta budget for W-16 that keeps total growth within
  R-W16-003, and (c) a reference measurement script committed for reproducibility
- **So that** S-9.01..S-9.07 each have an unambiguous per-plugin bundle size budget
  and a concrete measurement baseline to validate against at delivery time

## Goal

Establish the two enabling artifacts that all 7 batch stories (S-9.01..S-9.07)
depend on:

1. **Responsibility A (Bundle-size baseline):** Measure the current WASM plugin bundle
   sizes for the post-rc.4 release artifacts (after all 12 Tier 1 WASM ports from W-15
   are merged). Capture total bundle size (dispatcher binary + all hook-plugins/*.wasm)
   and per-plugin sizes. Commit results to `.factory/architecture/perf-baseline-w16.md`
   and update `.factory/measurements/E-8-bash-baseline.json` (or create a parallel
   `W-16-bundle-baseline.json`) with the W-16 starting point.

2. **Responsibility B (W-16 ceiling establishment):** Based on the bundle-size and
   cold-start latency measurements, establish the W-16 gate model per ADR-014 R-8.09
   revised (Amendment 2026-05-03):
   - **Primary gate (HARD):** cold-start p95 ≤ 500ms (inherited from S-8.00 / E-8 R-8.08).
   - **Advisory soft cap:** cumulative `all_hook_plugins_wasm_bytes` growth ≤ 100% vs
     v1.0.0-rc.1 pre-W-15 baseline (measured fresh per Task A.0). Target ceiling = 2 ×
     Task A.0's measured value. Applies to `all_hook_plugins_wasm_bytes`, NOT
     `grand_total_bytes` (dispatcher binary dominates at ~12MB). Note: the ~7.2MB
     figure from research §Q3 was a projection, not a measurement — discard it.
   - **Hard kill-switch:** 30MB cumulative bundle; crossing requires architecture review.
   - **Required per-wave telemetry:** `(bundle_size_delta_bytes, cold_start_p95_delta_ms)`.
   - **Pause criterion:** wave paused if cold-start regresses >10% vs previous wave.
   The revised model REPLACES E-8 R-8.09's simple 25% bundle-only ceiling. Original
   25% ceiling would have blocked W-16 even when cold-start budget is satisfied.

Both responsibilities are in this single story (not splittable; mirrors S-8.00 D-8).
