## Capability Anchor Justification

Anchoring to CAP-028: "Install and update the plugin via Claude Code marketplace" per capabilities.md §CAP-028.
Justification: S-4.08 is the 1.0.0-rc.1 release gate — it executes `bump-version.sh 1.0.0-rc.1`, which co-stamps plugin.json, marketplace.json, and CHANGELOG before the release bot commit. BC-9.01.001 (bump-version.sh accepts prerelease format) is the primary behavioral contract gating this story's version-bump step. Without this gate, the marketplace channel cannot receive a correctly labelled rc.1.
