## Part B — Observations (non-finding, below NITPICK)

### O-001 [LOW] — `decision-log-monotonic-rows.sh` regex `[\( ]` includes literal backslash

**Where:** `plugins/vsdd-factory/hooks/dim2-gates/decision-log-monotonic-rows.sh:61` (and echoed at line 58, README line 95).

**Observation:** The regex `^\| D-[0-9]+[\( ]` uses a bracket expression `[\( ]`. Inside `[...]`, the `\(` is two literal characters: `\` and `(`. The intent is "open-paren or space" — the `\` was likely added under the (incorrect) assumption that `(` needs escaping inside character classes. In practice no D-NNN row will end with `\`, so this never causes a false positive, but it's a regex hygiene drift worth a one-character fix to `[( ]` for POLICY 13 cleanliness. Same regex appears verbatim in the README's Script Details section, so a fix would propagate to the doc too.

**Severity:** LOW (cosmetic regex; no functional impact on any reachable input).

**Route:** in-scope cleanup — paired with fix-burst-2 implementer dispatch.

### O-002 [LOW] — `block-label-canonical-form.sh` echoes a synthetic alternation grep command but actually runs 9 per-label greps

**Where:** `plugins/vsdd-factory/hooks/dim2-gates/block-label-canonical-form.sh:96`.

**Observation:** The script prints `$ grep -E '^\*\*(Parent-commit|Adversary verdict|...)[: ]' ${BURST_LOG}` as the literal-shell attestation header, but internally executes 9 separate `grep -qE "^\*\*${LABEL}[: ]"` invocations in a loop (lines 84-94). Per POLICY 15 (verbatim stdout discipline) and D-449(a) (literal-shell-execution-evidence), the printed `$` line should reflect what was actually executed. A pasted Dim-2 attestation block citing this script's stdout would mislead a reader into thinking a single alternation grep ran. Per-label loop is the correct implementation choice (loop allows MISSING attribution); the fix is in the echoed prefix only — e.g., emit `$ for L in Parent-commit "Adversary verdict" ...; do grep -qE "^\*\*${L}[: ]" file; done` or simply omit the synthetic `$` echo.

**Severity:** LOW (semantic attestation drift, not behavioral). All other 10 scripts emit faithful `$` echoes (verified by reading each `echo "$ ..."` line vs subsequent execution).

**Route:** in-scope cleanup — paired with fix-burst-2 implementer dispatch.
