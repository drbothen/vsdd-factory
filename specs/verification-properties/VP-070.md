# VP-070: `validate-artifact-path` Path-Pattern Matching Is Pure and Deterministic

## Property Statement

The path-pattern matching function in `validate-artifact-path/src/lib.rs` is
**pure and deterministic**: for the same `(path, registry)` input pair, the
function MUST always return the same `MatchResult` (Block, Warn, Advisory, or
NoMatch) regardless of:

- Call order
- Thread identity
- Wall-clock time
- Any external state not present in the input arguments

Formally:

```
∀ path: &str, registry: &PathRegistry:
    matches_canonical(path, registry) == matches_canonical(path, registry)
```

Additionally, the following safety invariants hold unconditionally:

1. **Non-`.factory/` paths always return NoMatch (early exit).** The hook has no
   opinion on writes outside `.factory/`.
2. **Empty path always returns NoMatch.** Empty paths cannot match any `.factory/`
   prefix.
3. **Block decision is never returned for non-`.factory/` paths.** No combination
   of registry entries can cause a non-`.factory/` path to be blocked.
4. **The decision function is total.** It terminates for all inputs and never
   panics.

## Formal Property Statement (kani harness)

```rust
#[cfg(kani)]
mod kani_proofs {
    use super::*;

    /// Determinism: same (path, registry) always yields same decision.
    /// Kani verifies this holds for all inputs up to the bounded string length.
    #[kani::proof]
    #[kani::unwind(16)]
    fn proof_matches_canonical_is_deterministic() {
        // Kani constructs arbitrary bounded inputs
        let path: String = kani::any();
        kani::assume(path.len() <= 64);

        // Use a fixed-size registry for bounded verification
        let registry = PathRegistry::fixture_single_entry_block();

        let decision_1 = matches_canonical(&path, &registry);
        let decision_2 = matches_canonical(&path, &registry);

        kani::assert(
            decision_1 == decision_2,
            "matches_canonical must be deterministic: same inputs must yield same decision"
        );
    }

    /// Non-.factory/ paths must always yield NoMatch (never Block).
    /// "Non-.factory/" means neither a relative .factory/ path (starts_with(".factory/"))
    /// nor an absolute path whose components include .factory/ (contains("/.factory/")).
    /// After 8b4f697f introduced absolute-path matching, the prior assumption
    /// `!starts_with(".factory/")` was insufficient — an absolute path like
    /// `/abs/proj/.factory/specs/foo.md` passed the assume but produced Block, not NoMatch.
    /// (F-P19-002)
    #[kani::proof]
    #[kani::unwind(16)]
    fn proof_non_factory_path_always_nomatch() {
        let path: String = kani::any();
        kani::assume(path.len() <= 64);
        // Exclude both relative and absolute .factory/ paths so the proof
        // covers genuinely out-of-scope inputs only.
        kani::assume(
            !path.starts_with(".factory/")
                && !path.contains("/.factory/"),
        );

        let registry = PathRegistry::fixture_single_entry_block();
        let decision = matches_canonical(&path, &registry);

        kani::assert(
            matches!(decision, MatchResult::NoMatch),
            "genuinely out-of-scope path must always yield NoMatch"
        );
    }

    /// Empty path must yield NoMatch.
    #[kani::proof]
    fn proof_empty_path_is_nomatch() {
        let registry = PathRegistry::fixture_single_entry_block();
        let decision = matches_canonical("", &registry);
        kani::assert(
            matches!(decision, MatchResult::NoMatch),
            "empty path must yield NoMatch"
        );
    }

    /// Block decision only fires for .factory/ paths with enforcement_level=block.
    #[kani::proof]
    #[kani::unwind(16)]
    fn proof_block_only_on_factory_path_with_block_level() {
        let path: String = kani::any();
        kani::assume(path.len() <= 64);

        let registry = PathRegistry::fixture_advisory_only();
        let decision = matches_canonical(&path, &registry);

        // Registry with only advisory entries must never produce Block
        kani::assert(
            !matches!(decision, MatchResult::Block),
            "advisory-only registry must never produce Block for any path"
        );
    }
}
```
