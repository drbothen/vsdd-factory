# VP-071: `validate-per-story-adversary-convergence` Block Invariant

## Property Statement

The `validate-per-story-adversary-convergence` WASM hook MUST emit a block
outcome (advisory-block-mode: `{"outcome":"block",...}` to stdout with
`hook.block` event) whenever ANY of the following conditions holds for any
story in the wave under review:

1. **Missing state file:** The convergence state file at
   `.factory/cycles/<cycle-id>/<story-id>/adversary-convergence-state.json`
   does not exist (read returns `None`).
2. **Insufficient passes:** The state file exists and parses correctly, but
   `passes_clean < 3`.
3. **Non-nitpick classification:** The state file exists and parses correctly,
   but `last_classification != "NITPICK_ONLY"` (including `null` as a
   non-nitpick value).

The hook MUST return `HookResult::Continue` in all cases (advisory-block-mode
per HOST_ABI.md §Advisory block-mode pattern). The block signal is conveyed
via stdout JSON, not via `HookResult::Block`.

Conversely (the happy-path invariant), the hook MUST NOT emit a block signal
when ALL stories satisfy BOTH of:
- `passes_clean >= 3`
- `last_classification == "NITPICK_ONLY"`

## Formal Property Statement (kani harness)

```rust
#[cfg(kani)]
mod kani_proofs {
    use super::*;

    /// Block invariant — Part A: missing state file always blocks.
    /// Regardless of how many stories are in the wave, a missing file
    /// for ANY story causes a block.
    #[kani::proof]
    fn proof_missing_state_file_always_blocks() {
        // Single-story case: state file absent
        let state: Option<ConvergenceState> = None;
        let outcome = evaluate_convergence(state);
        kani::assert(
            matches!(outcome, ConvergenceOutcome::Block(_)),
            "missing state file must always produce Block outcome"
        );
    }

    /// Block invariant — Part B: passes_clean < 3 always blocks.
    #[kani::proof]
    #[kani::unwind(4)]
    fn proof_insufficient_passes_always_blocks() {
        let passes: u32 = kani::any();
        kani::assume(passes < 3);

        let state = Some(ConvergenceState {
            passes_clean: passes,
            last_classification: Some("NITPICK_ONLY".to_string()),
            last_finding_count: Some(0),
            last_timestamp: None,
            deferred_findings: vec![],
        });
        let outcome = evaluate_convergence(state);
        kani::assert(
            matches!(outcome, ConvergenceOutcome::Block(_)),
            "passes_clean < 3 must always produce Block outcome"
        );
    }

    /// Block invariant — Part C: non-NITPICK_ONLY classification always blocks
    /// even when passes_clean >= 3.
    #[kani::proof]
    fn proof_non_nitpick_classification_always_blocks() {
        // passes_clean >= 3 but classification is not NITPICK_ONLY
        let state = Some(ConvergenceState {
            passes_clean: 3,
            last_classification: Some("HIGH".to_string()),
            last_finding_count: Some(2),
            last_timestamp: None,
            deferred_findings: vec![],
        });
        let outcome = evaluate_convergence(state);
        kani::assert(
            matches!(outcome, ConvergenceOutcome::Block(_)),
            "non-NITPICK_ONLY classification must produce Block even when passes_clean >= 3"
        );
    }

    /// Block invariant — Part D: null/None classification always blocks.
    #[kani::proof]
    fn proof_null_classification_blocks() {
        let state = Some(ConvergenceState {
            passes_clean: 5,
            last_classification: None,  // null in JSON
            last_finding_count: None,
            last_timestamp: None,
            deferred_findings: vec![],
        });
        let outcome = evaluate_convergence(state);
        kani::assert(
            matches!(outcome, ConvergenceOutcome::Block(_)),
            "null/None last_classification must produce Block"
        );
    }

    /// Happy path: passes_clean >= 3 AND last_classification == NITPICK_ONLY
    /// must produce Continue (no block).
    #[kani::proof]
    #[kani::unwind(4)]
    fn proof_converged_story_produces_continue() {
        let passes: u32 = kani::any();
        kani::assume(passes >= 3);

        let state = Some(ConvergenceState {
            passes_clean: passes,
            last_classification: Some("NITPICK_ONLY".to_string()),
            last_finding_count: Some(0),
            last_timestamp: None,
            deferred_findings: vec![],
        });
        let outcome = evaluate_convergence(state);
        kani::assert(
            matches!(outcome, ConvergenceOutcome::Continue),
            "converged story (passes >= 3, NITPICK_ONLY) must produce Continue"
        );
    }

    /// Hook always returns HookResult::Continue regardless of block signal
    /// (advisory-block-mode invariant).
    #[kani::proof]
    fn proof_hook_result_is_always_continue() {
        // Both blocking and non-blocking paths must return HookResult::Continue
        let state_missing: Option<ConvergenceState> = None;
        let state_converged = Some(ConvergenceState {
            passes_clean: 3,
            last_classification: Some("NITPICK_ONLY".to_string()),
            last_finding_count: Some(0),
            last_timestamp: None,
            deferred_findings: vec![],
        });

        let result_block = hook_result_for(state_missing);
        let result_continue = hook_result_for(state_converged);

        kani::assert(
            matches!(result_block, HookResult::Continue),
            "hook must return Continue even when emitting a block signal (advisory-block-mode)"
        );
        kani::assert(
            matches!(result_continue, HookResult::Continue),
            "hook must return Continue on happy path"
        );
    }
}
```
