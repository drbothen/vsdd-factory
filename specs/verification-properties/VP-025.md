# VP-025: Host Function ABI Surface Is Complete and Stable

## Property Statement

setup_linker registers every named host function in the vsdd namespace: {log, emit_event, read_file, exec_subprocess, session_id, dispatcher_trace_id, plugin_root, plugin_version, cwd, env}. A WAT module that imports all vsdd functions links and executes successfully. No host function has been silently removed.

## Source Contract

- **BC:** BC-1.05.033 — setup_linker_registers_every_vsdd_import
- **Postcondition/Invariant:** SDK ABI surface is stable — no silent function removal

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| integration | wasmtime linker lookup | yes — fixed 10-function set | all vsdd namespace imports |

## Proof Harness Skeleton

```rust
// Integration test: crates/factory-dispatcher/tests/host_functions.rs
#[test]
fn verify_vp_025_host_abi_complete() {
    let engine = Engine::default();
    let mut store = Store::new(&engine, ());
    let linker = setup_linker(&engine).unwrap();

    let vsdd_fns = ["log", "emit_event", "read_file", "exec_subprocess",
                    "session_id", "dispatcher_trace_id", "plugin_root",
                    "plugin_version", "cwd", "env"];

    for name in &vsdd_fns {
        assert!(
            linker.get(&mut store, "vsdd", name).is_some(),
            "missing vsdd::{}", name
        );
    }
}
```
