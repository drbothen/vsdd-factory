# VP-062: Count Propagation Invariant — validate-count-propagation.sh Flags Drift and Is Correctly Registered

## Property Statement

After delivery of S-7.02: (1) `validate-count-propagation.sh` correctly exits non-zero when a count-keyword pair appears at different values in two or more sibling index files, exits zero when counts are consistent or absent, and runs in under 200ms on a typical corpus; (2) the hook is registered in `hooks-registry.toml` as a PostToolUse hook on Write/Edit to the defined index files; (3) `plugins/vsdd-factory/rules/lessons-codification.md` exists with the required `[process-gap]` tag definition and three-step protocol; (4) `validate-template-compliance.sh` enforces the VP multi-BC `source_bc:` convention.

## Source Contract

- **Primary: BC-7.05.001** — validate-count-propagation.sh drift detection and exit semantics
- **BC-5.37.001** — state-manager defensive sweep protocol
- **BC-5.37.002** — sweep result logging requirement
- **BC-7.05.002** — hook performance and determinism (under 200ms)
- **BC-7.05.003** — VP multi-BC source_bc compliance check
- **BC-8.28.001** — lessons-codification.md rule file existence and content
- **BC-8.28.002** — orchestrator checklist references lessons-codification.md
- **BC-9.02.001** — hooks-registry.toml registration of validate-count-propagation

## Proof Harness Skeleton

```bash
# Integration test: plugins/vsdd-factory/hooks/tests/test_count_propagation.sh

# Fixture setup
mkdir -p /tmp/count-prop-test
cp .factory/STATE.md /tmp/count-prop-test/STATE.md
cp .factory/specs/behavioral-contracts/BC-INDEX.md /tmp/count-prop-test/BC-INDEX.md

# Test 1: drift detected exits non-zero
echo "1,875 BCs in total" > /tmp/count-prop-test/STATE.md
echo "1,863 BCs in total" > /tmp/count-prop-test/BC-INDEX.md
plugins/vsdd-factory/hooks/validate-count-propagation.sh /tmp/count-prop-test/STATE.md
assert_exit_nonzero $?
# stderr must contain "COUNT DRIFT DETECTED"

# Test 2: consistent counts exit zero
echo "1,875 BCs in total" > /tmp/count-prop-test/STATE.md
echo "1,875 BCs in total" > /tmp/count-prop-test/BC-INDEX.md
plugins/vsdd-factory/hooks/validate-count-propagation.sh /tmp/count-prop-test/STATE.md
assert_exit_zero $?

# Test 3: count absent from sibling exits zero
echo "1,875 BCs in total" > /tmp/count-prop-test/STATE.md
echo "No BC mentions here" > /tmp/count-prop-test/BC-INDEX.md
```
