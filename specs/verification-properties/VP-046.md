# VP-046: All hooks-registry.toml Entries Correspond to Registered Hook Scripts

## Property Statement

Every entry in hooks-registry.toml has a corresponding .sh file under plugins/vsdd-factory/hooks/. No registry entry references a missing script. The generate-registry-from-hooks-json.sh script is idempotent: re-running it on a registry produces an identical file.

## Source Contract

- **BC:** BC-7.01.001 — All registered hooks fire via legacy-bash-adapter
- **Postcondition/Invariant:** DI-014 — schema version + registry entries must be consistent

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| manual | Shell script or CI check | no | registry ↔ script file correspondence |

## Proof Harness Skeleton

```bash
# CI check: every script_path in hooks-registry.toml must exist
while IFS= read -r script_path; do
    if [[ ! -f "plugins/vsdd-factory/$script_path" ]]; then
        echo "MISSING: $script_path"
        exit 1
    fi
done < <(grep 'script_path' plugins/vsdd-factory/hooks-registry.toml | awk -F'"' '{print $2}')
```
