# VP-057: Adversarial Review Convergence — Mis-Anchoring Always Blocks, 3-Clean-Pass Minimum

## Property Statement

The adversary agent: (1) blocks convergence whenever it detects a mis-anchored artifact (BC ID that doesn't exist, referenced module not in architecture), (2) requires a minimum of 3 clean passes before declaring convergence, (3) cannot see prior adversarial review outputs (information wall), and (4) dispatches against a SHA-current factory-artifacts state (adversary SHA-currency gate).

- **BC:** BC-5.04.003 — adversary: mis-anchoring always blocks convergence
