# VP-009: prune_old Removes Only Dispatcher-Internal Files Older Than Threshold

## Property Statement

prune_old(N) removes only files matching the pattern dispatcher-internal-*.jsonl that are older than N days. It does not remove other files in the log directory, does not remove files newer than the threshold, and is a no-op when the log directory does not exist.

## Source Contract

- **BC:** BC-1.06.004 — prune_old removes only dispatcher-internal-*.jsonl files older than threshold
- **Postcondition/Invariant:** DI-009 — Internal logs are pruned to 30 days at dispatcher start

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| unit-test | Rust unit test with tempdir | yes — mixed file set | filename filter + age filter + missing dir |

## Proof Harness Skeleton

```rust
#[test]
fn verify_vp_009_prune_only_matching_old_files() {
    let dir = tempdir();

    // Old matching file (should be pruned)
    write_file(&dir, "dispatcher-internal-2026-01-01.jsonl", old_mtime);
    // Recent matching file (should be kept)
    write_file(&dir, "dispatcher-internal-2026-04-24.jsonl", recent_mtime);
    // Non-matching file (should always be kept)
    write_file(&dir, "other-file.log", old_mtime);

    prune_old(&dir, 30);

    assert!(!dir.join("dispatcher-internal-2026-01-01.jsonl").exists());
    assert!(dir.join("dispatcher-internal-2026-04-24.jsonl").exists());
    assert!(dir.join("other-file.log").exists());
}
```
