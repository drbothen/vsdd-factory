# Behavioral Contract BC-1.05.022: factory-dispatcher::host::read_file::reads_allowed_file — file under path_allow with size <= max_bytes returns its contents

## Description

When a file `ok.txt` ("hello world") is under `path_allow` with `ctx.plugin_root = tempdir`, `prepare(&ctx, file_path, 1024)` returns `Ok((b"hello world", _))`. Allow-listed paths can be read up to `max_bytes`.

## Preconditions

1. Tempdir has `ok.txt` with contents `"hello world"`.
2. Capability declares `path_allow = [tempdir]`.
3. `ctx.plugin_root = tempdir`.

## Postconditions

1. `prepare(&ctx, file_path, 1024)` returns `Ok((b"hello world", _))`.

## Invariants

1. Files under `path_allow` and within `max_bytes` are readable.
