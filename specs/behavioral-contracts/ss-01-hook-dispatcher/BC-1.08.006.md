# Behavioral Contract BC-1.08.006: dispatcher projects whole process env into env_view (capability gate enforced at host fn call time)

## Description

At dispatcher startup, `base_host_ctx.env_view = std::env::vars().collect()` — the whole process environment is projected. The per-plugin `env_allow` gate is enforced inside the `vsdd::env` host fn, NOT at projection time.

## Preconditions

1. Dispatcher startup.

## Postconditions

1. `base_host_ctx.env_view` contains the full process env (Map<String, String>).
2. Per-plugin `env_allow` enforcement happens at host-fn call time, not at projection time.

## Invariants

1. Capability filtering is at access time, not at projection time.
