# Behavioral Contract BC-10.04.001: factory-obs: lifecycle manager for the local observability docker-compose stack

> Section: factory-obs CLI
> Source BC (audit ID): BC-AUDIT-2117

## Description

Wraps `docker compose` against `tools/observability/docker-compose.yml`. Stack: otel-collector + Loki + Prometheus + Grafana + renderer. v0.78.0+ multi-factory model: a registry at `~/.config/vsdd-factory/watched-factories` lists absolute paths; `factory-obs up` generates `docker-compose.override.yml` with one bind mount per registered factory.
