# Behavioral Contract BC-10.01.003: compute-input-hash: input resolution against ARTIFACT_DIR + .factory/ search bases

> Section: compute-input-hash CLI
> Source BC (audit ID): BC-AUDIT-2102

## Description

Resolves each input file against ordered search bases: `ARTIFACT_DIR`, `ARTIFACT_DIR/..`, `${FACTORY_ROOT}/specs`, `${FACTORY_ROOT}/stories`, `${FACTORY_ROOT}/phase-0-ingestion`, `${FACTORY_ROOT}/holdout-scenarios`, `${FACTORY_ROOT}`. Strips `.factory/` prefix when FACTORY_ROOT is detected. Glob inputs (`pattern/**` or trailing `/`) expand via `find ... -name '*.md'` sorted with `LC_ALL=C` for determinism.
