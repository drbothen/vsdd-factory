# Behavioral Contract BC-10.06.003: factory-replay: pairing rule — sort by ts_epoch, group by session_id, latest events at top

> Section: factory-replay CLI
> Source BC (audit ID): BC-AUDIT-2131

## Description

`sessions` builds `(session_id, count, first_ts, last_ts)` rows via awk over the JSONL stream, then sorts by last_ts descending. `show` sorts events for the session by `ts` ascending and renders TSV with cleaned-up timestamps (drops microseconds/tz suffix).
