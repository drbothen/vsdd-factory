# Behavioral Contract BC-2.04.002: hook-sdk::payload::posttooluse_payload_with_response — SDK payload includes typed access to tool_response.exit_code

## Description

A PostToolUse envelope with `tool_response={exit_code:0}` parses such that `tool_response.get("exit_code").as_i64() == Some(0)`. Plugin authors can downcast `tool_response` to typed values without manual JSON walking.

## Preconditions

1. Envelope has `tool_response: {exit_code: 0}`.

## Postconditions

1. After parsing, `tool_response.get("exit_code").as_i64() == Some(0)`.

## Invariants

1. Typed access to `tool_response` fields is supported.
