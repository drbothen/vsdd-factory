# Behavioral Contract BC-1.05.013: factory-dispatcher::host::emit_event::decode_single_pair — length-prefixed key/value buffer with one pair round-trips through decode_fields

## Description

A buffer encoding `("k", "v")` as `[1u32_LE, 'k', 1u32_LE, 'v']` decodes via `decode_fields(&buf)` into `Ok(vec![("k".to_string(), "v".to_string())])`. SDK-side `encode_fields` and dispatcher-side `decode_fields` are byte-compatible.

## Preconditions

1. Buffer encodes one (k,v) pair using length-prefix layout.

## Postconditions

1. Returns `Ok(vec![("k", "v")])`.

## Invariants

1. Single-pair round-trip is byte-stable cross-crate.
