# Behavioral Contract BC-1.05.028: factory-dispatcher::host::exec_subprocess::binary_allow_matches_basename — allow-list compares against basename, not full path

## Description

`binary_allowed("/usr/bin/git", &["git"])` and `binary_allowed("git", &["git"])` both return `true`; `binary_allowed("curl", &["git"])` returns `false`. Allow-list match is on the file basename — operators can declare `binary_allow = ["git"]` regardless of where git is installed.

## Preconditions

1. `binary_allowed(cmd, allow_list)` invoked with various cmd shapes.

## Postconditions

1. Match is on file basename of `cmd`.
2. Returns true iff basename is in `allow_list`.

## Invariants

1. Allow-list is path-independent — basename only.
