# Step 7: Refactor (if needed)

With all tests green, refactor for code quality while maintaining test coverage.

## Inputs

- All tests passing (new + regression)
- Implementation from Step 4

## Actions

1. Extract common patterns
2. Improve naming
3. Reduce duplication between new and existing code
4. Re-run ALL tests after each refactor step
5. Never refactor without test coverage -- every refactor must be validated

## Outputs

- Refactored code (if changes were made)
- Confirmation that all tests still pass after refactoring

## Completion Criteria

- Code quality improved (or "no refactoring needed" documented)
- All tests (new + regression) still pass after every refactor step
