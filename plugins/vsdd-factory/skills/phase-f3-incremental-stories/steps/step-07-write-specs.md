# Step 7: Write Story Specs

Write finalized story specs to the factory directory and update the story index.

## Inputs

- Finalized story specs from Steps 2-6
- Story index at `.factory/stories/story-index.md`

## Actions

1. Write each new story to `.factory/phase-f3-stories/STORY-XXX-description.md`
2. Update the story index at `.factory/stories/story-index.md` to include new stories
3. Include in each story spec: frontmatter, requirements, ACs, dependencies, estimates

## Outputs

- `.factory/phase-f3-stories/STORY-XXX-description.md` (one per new story)
- `.factory/phase-f3-stories/dependency-graph-extended.md`
- `.factory/phase-f3-stories/conflict-report.md` (if conflicts found)
- Updated `.factory/stories/story-index.md`

## Completion Criteria

- All story specs are written with complete content
- Story index is updated with new entries
- Dependency graph visualization is generated
