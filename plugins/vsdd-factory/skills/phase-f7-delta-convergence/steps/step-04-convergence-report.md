# Step 4: Delta Convergence Report

Write the comprehensive convergence report with all metrics.

## Inputs

- Convergence metrics from Step 1
- Regression validation from Step 2
- Traceability chain from Step 3

## Actions

1. Write the convergence report to `.factory/phase-f7-convergence/delta-convergence-report.md`
2. Include:
   - Feature summary (request, spec version change, stories, files)
   - Five-dimensional convergence table with targets, actuals, and status
   - Regression validation table (baseline vs current)
   - Traceability chain links
   - Recommendation: READY FOR MERGE or NEEDS WORK (with specific issues)

## Outputs

- `.factory/phase-f7-convergence/delta-convergence-report.md`

## Completion Criteria

- All metrics are populated (no empty cells)
- Recommendation is clear and justified
- Report follows the template format from the SKILL.md
