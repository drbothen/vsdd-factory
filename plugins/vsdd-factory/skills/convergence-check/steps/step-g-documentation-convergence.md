# Step G: Documentation Convergence

> **Shared context:** Read `./_shared-context.md` before executing this step.

## Procedure

1. Verify CLAUDE.md updated with current architecture
2. Verify API docs generated (if applicable)
3. Verify README reflects current state
4. Check for stale references to removed features

## Pass Criteria

- Docs match implementation
- No stale references
- README reflects current functionality

## Output

Update the Documentation row in the convergence report summary table. Write detail section under `## 7. Documentation Convergence`.

After all 7 dimensions are assessed, write the **Overall** verdict (CONVERGED or NOT CONVERGED) and the **Remaining Items** section listing any failing dimensions with severity and estimated effort.
